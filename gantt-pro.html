Quick Guide

Everything you need to know to get started

使用教程

快速上手甘特图工具

Adding Tasks
  • Click Add Task in the left panel
  • Set a name, start date, and end date
  • Use +1 day / +1 week / +1 month shortcuts to set the end date quickly
  • Double-click any bar in the chart to edit it
Sections & Groups
  • Click + Section to create a named group (e.g. "Preparation")
  • Drag any task card to move it into a different section
  • Click the section header to collapse / expand it
  • Rename or delete sections with the icons on the right
Progress & Milestones
  • Drag the Progress slider (0–100%) when editing a task
  • The bar fills in and shows the percentage inside
  • At 100% a ✓ badge appears; overdue tasks show a red dashed border
  • Enable Milestone for a single-date diamond marker
Dependencies
  • In the task editor, tick tasks under Dependencies
  • A dashed arrow connects predecessor to dependent task in the chart
  • Circular dependencies are detected and blocked automatically
Navigation & Zoom
  • Click TODAY to jump the chart to the current date
  • Use + / − buttons (or keyboard) to zoom in/out
  • Five zoom levels: Month · 2 Weeks · Standard · Week · Day
  • Scroll horizontally to pan the timeline
Save & Export
  • Changes are auto-saved to the browser every 0.5 s
  • Use Save to download a .gantt.json backup
  • Use Load to open a saved file on any computer
  • PNG / PDF exports the full chart as an image
⌨ Keyboard shortcuts  —  Ctrl Z Undo   Ctrl Y Redo   + Zoom in   − Zoom out   Esc Close modal

💾 Important: Auto-save stores data in this browser only. If you switch browsers or computers, use Save to export a JSON file first, then Load it on the new machine.
添加任务
  • 点击左侧面板的 Add Task 按钮
  • 填写任务名称、开始日期和结束日期
  • 使用 +1天 / +1周 / +1月 快捷按钮快速设置结束日期
  • 在图表中双击任意进度条可直接编辑该任务
分组管理
  • 点击 + Section 创建自定义分组（如"前期准备"、"项目推进"）
  • 直接拖拽任务卡片到其他分组
  • 点击分组标题可折叠 / 展开该组
  • 点击右侧图标可重命名或删除分组
进度与里程碑
  • 编辑任务时拖动进度滑块（0–100%）
  • 进度条会填充，并在条内显示完成百分比
  • 完成后出现 ✓ 标记；逾期未完成任务显示红色虚线边框
  • 勾选 Milestone 可创建单日期的菱形里程碑标记
前置任务（依赖）
  • 在任务编辑页面勾选 Dependencies 中的前置任务
  • 图表中会显示连接前后任务的虚线箭头
  • 系统自动检测并阻止循环依赖
导航与缩放
  • 点击 TODAY 按钮跳转到当前日期
  • 点击 + / − 按钮（或键盘）调整缩放级别
  • 共五个级别：月览 · 双周 · 标准 · 周视图 · 日视图
  • 横向滚动可平移时间轴
保存与导出
  • 每次改动后 0.5 秒自动保存到当前浏览器
  • 点击 Save 下载 .gantt.json 备份文件
  • 点击 Load 在任意电脑上打开已保存文件
  • PNG / PDF 将完整图表导出为图片
⌨ 键盘快捷键  —  Ctrl Z 撤销   Ctrl Y 重做   + 放大   − 缩小   Esc 关闭弹窗

💾 注意：自动保存仅在当前浏览器有效。换电脑或浏览器前，请先点击 Save 导出 JSON 文件，再通过 Load 在新设备上恢复。
